
# 🐧 Penguin2D

Projeto de estudo em desenvolvimento de jogos 2D utilizando a engine Godot e a linguagem GDScript.

## 🎮 Sobre o Projeto

O Penguin2D é um jogo 2D desenvolvido com foco em aprendizado e evolução prática no universo de game development.
O projeto explora mecânicas clássicas de jogos de plataforma, movimentação avançada, inimigos com IA simples, física, animações e arquitetura de sistemas.

Além de servir como prática de programação, o projeto também está sendo utilizado para aprofundar conhecimentos em:

* Desenvolvimento de jogos 2D
* GDScript
* Física e colisões
* Estruturação de código
* Máquinas de estado (State Machine)
* Level Design
* Sistemas de câmera e parallax

---

# 🚀 Tecnologias Utilizadas

* Godot Engine 4
* GDScript
* TileMap
* Parallax2D
* Tween System
* AnimatedSprite2D
